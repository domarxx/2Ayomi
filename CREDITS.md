# Créditos e Atribuições (Credits & Attributions)

Este projeto, **AniyomiSY**, é um aplicativo híbrido que combina recursos e funcionalidades de dois forks notáveis do ecossistema original do Tachiyomi.

Agradecemos imensamente aos desenvolvedores e colaboradores dos seguintes projetos originais:

1. **Aniyomi**
   * **Repositório:** [https://github.com/aniyomiorg/aniyomi](https://github.com/aniyomiorg/aniyomi)
   * **Descrição:** Base principal deste projeto, adicionando suporte a anime, player de vídeo nativo, rastreadores e extensões de anime.
   * **Licença:** Apache License 2.0

2. **TachiyomiSY**
   * **Repositório:** [https://github.com/jobobby04/TachiyomiSY](https://github.com/jobobby04/TachiyomiSY)
   * **Descrição:** Fonte de recursos avançados de usabilidade de mangá, incluindo busca inteligente, filtros salvos, recomendações e melhorias de UI baseadas no TachiyomiAZ/Neko.
   * **Licença:** Apache License 2.0

3. **Tachiyomi (Projeto Original)**
   * **Repositório:** [https://github.com/tachiyomiorg/tachiyomi](https://github.com/tachiyomiorg/tachiyomi) (Arquivado)
   * **Descrição:** O leitor de mangás de código aberto para Android que serviu como fundação e inspiração para todo este ecossistema.
   * **Licença:** Apache License 2.0
